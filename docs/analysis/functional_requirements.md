# Requirements

Create a web application that implements an information system for individual entities of specific categories.

**Design**: At the developer’s discretion.

**Core functionality:**
- Information about the entities must be stored in a database.
- Data related to the entities (including numerical values) must be entered by the user of the application.

**Additional features:**
- Implement the ability for the user of the application to query information about an entity and display it.
- Provide functionality to view some statistical information about entities based on their stored values.
- Export entities with their associated data to a external file.
